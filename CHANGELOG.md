# Changelog

All notable changes to Evictor are documented here.

## Unreleased

### Features

- Added the initial Go API and Next.js dashboard monorepo foundation.

- Added the planned backend package boundaries for handlers, services, storage,
  metrics, providers, workers, primitives, models, and libraries.

### Fixes

- Generate Next.js route types before strict typechecking.
- Handle rewritten commit ranges in CI after feature-branch force pushes.
